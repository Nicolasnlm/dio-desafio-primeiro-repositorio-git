# Bluetooth

O Bluetooth faz uma conexão ponto a ponto, sem depender do uso da Internet, existem 3 Classes de conexão:



| Classe | Potência Max. | Alcance |
| :----: | :-----------: | :-----: |
|   1    |     100mW     |  100m   |
|   2    |     2,4mW     |   10m   |
|   3    |      1mW      |   1m    |

#### Versões do Bluetooth

| Versão      | Taxa de Transmissão |
| :---------- | :-----------------: |
| **1.2**     |       1 Mbps        |
| **2.0+EDR** |       3 Mbps        |
| **3.0**     |       24 Mbps       |
| **4.0**     |       25 Mbps       |
| **5.0**     |       50 Mbps       |
